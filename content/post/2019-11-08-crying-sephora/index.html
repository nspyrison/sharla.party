library(tidytext)

crying_tokens <- crying %>%
  mutate(review_id = row_number()) %>%
  mutate(review = paste(review_title, review_body)) %>%
  select(review_id, stars, review) %>%
  unnest_tokens(word, review)

Next, I‚Äôll get rid of stop words, and with the remaining words, count how many times they appear:

crying_tokens <- crying_tokens %>%
  anti_join(stop_words, by = "word")

crying_tokens_frequency <- crying_tokens %>%
  anti_join(stop_words, by = "word") %>%
  count(word, sort = TRUE)

I also want to look at the sentiment of each review compared to the star rating. I‚Äôm guessing they won‚Äôt really match up because this context is‚Ä¶ hard to capture. But let‚Äôs see. I‚Äôm using the NRC sentiment lexicon because it has the most words in common with my data set. The net sentiment is the number of words with positive sentiment minus the number with negative sentiment.

crying_sentiment <- crying_tokens %>%
  inner_join(get_sentiments("nrc") %>%
    filter(sentiment %in% c("positive", "negative")), by = "word") %>%
  count(review_id, stars, sentiment) %>%
  spread(sentiment, n, fill = 0) %>%
  mutate(net_sentiment = positive - negative)

ggplot(
  crying_sentiment,
  aes(x = stars, y = net_sentiment)
) +
  geom_jitter(width = 0.1, height = 0.1, alpha = 0.5) +
  labs(
    x = "Star Rating",
    y = "Net Sentiment"
  )
